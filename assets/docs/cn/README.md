## 如何安装 `Leanote`

安装 `Leanote` 二进制和源码的区别: 二进制版是编译好的 `Leanote`, 不用安装开发环境. `Leanote` 源码安装, 需要安装编译环境 `Golang`.

**注意:** 🔔⚠️ ⚠️ ⚠️ ⚠️  强烈建议使用Linux 或 Mac安装Leanote. 在Windows上, Leanote没有足够的测试, 所以不保证能在Windows上正常运行.
